## Stored outlines

Put detailed course outlines in this folder so they can be reused later (e.g., to generate flashcards, practice questions, summaries) without needing to re-paste them in chat.

### Convention
- One file per course, e.g. `ap-gov.md`, `apush.md`, `ap-world.md`, `ap-hug.md`
- Prefer headings by unit and consistent bullet formatting
